Qué es una excepción?

Una excepción es un error que ocurre durante la ejecución de un programa. Si no se maneja, el programa se detiene.

Estructura

    try { // código que puede fallar }

      catch (Exception e) { // qué hacer si ocurre }

        finally { // siempre se ejecuta }

Ejemplo

    try { int resultado = 10 / 0; }

      catch (ArithmeticException e)
      { System.out.println("No se puede dividir entre cero."); }

Ventajas

  • ⚡ Evitan que el programa se detenga
  • 🛠️ Facilitan la depuración
  • 🔒 Hacen el software más seguro
